## Sobre

Luminare Odontologia é uma Landing Page premium desenvolvida com foco em alta performance, estética "high-end" e experiência do usuário (UX) refinada. O projeto utiliza uma arquitetura puramente estática para garantir tempos de carregamento instantâneos e máxima compatibilidade.

O design implementa conceitos modernos como Glassmorphism, CSS Grid assimétrico e animações baseadas em scroll (Intersection Observer), sem depender de frameworks pesados.

## Tecnologias

*   **HTML5 Semântico**: Estruturação acessível e otimizada para SEO.
*   **CSS3 Moderno**: Variáveis CSS, Grid Layout, Flexbox, Scroll Snap e Animações.
*   **JavaScript (ES6+)**: Lógica "Vanilla" para interações complexas (Slider Antes/Depois, Tabs, Scroll Reveal) sem dependências externas.
*   **Performance First**: Zero bloated libraries (jQuery, Bootstrap, Tailwind removidos em favor de código nativo otimizado).

## Instalação Local

Para rodar o projeto localmente, você pode usar qualquer servidor estático (como Live Server do VSCode) ou via Python/Node:

### Opção 1: VSCode Live Server
1.  Instale a extensão "Live Server".
2.  Clique com botão direito em `index.html` e selecione "Open with Live Server".

### Opção 2: Python Simple HTTP Server
```bash
python -m http.server 8000
# Acesse http://localhost:8000
```

### Opção 3: Node `http-server`
```bash
npx http-server .
# Acesse o endereço indicado no terminal
```
